.. _gwc_rest:

GeoWebCache REST API
====================

This section will discuss the GeoWebCache REST API.

Note that for the GeoWebCache version integrated with GeoServer, the GWC REST API base URL is ``<geoserver base URL>/gwc/rest``, whilst
for the standalone GeoWebCache it's  jsut ``<geowebcache base URL>/rest``.
